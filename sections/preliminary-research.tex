 Numerous similar projects with alike problems can be found, though there are some more noticeable and important.

 In 2017, Google introduced federated learning while implementing the ``Next word suggestion'' feature on their Gboard. They took user interaction as their input and trained an artificial model to predict users next word. Without this innovation, all the private typing data from millions of devices would have to be transferred to a central server. At this point, we can easily understand Googles main concerns of standard data pipeline: Bandwidth usage and latency.
